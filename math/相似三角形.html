相似三角形

比例线段​

对于四条线段 aaa，bbb，ccc，ddd 如果两条线段的比（即它们长度的比）与另两条线段的比相等，如 ab=cd(ad=bc)\frac{a}{b} = \frac{c}{d} (ad=bc)ba​=dc​(ad=bc)，我们就说这四条线段成比例。

比例性质​

ab=cd⇔ac=bd\frac{a}{b} = \frac{c}{d} ⇔ \frac{a}{c} = \frac{b}{d}ba​=dc​⇔ca​=db​ ab=bc→b2=ac\frac{a}{b} = \frac{b}{c} → b^2 = acba​=cb​→b2=ac

比例中项​

如果 aaa、bbb、ccc 三个量成比例即 a:b=b:ca:b = b:ca:b=b:c，bbb 叫做 aaa 和 ccc 的比例中项。

例子​

1:2=2:41:2=2:41:2=2:4

222 是 111 和 444 的比例中项。

平行线分线段成比例​

性质​

两条直线被一组平行线所截，所得的对应线段成比例。

几何语言​

∵l1//l2//l3∵ l_1 // l_2 // l_3∵l1​//l2​//l3​ ∴ABBC=DEEF（上比下 = 上比下）∴\frac{AB}{BC} = \frac{DE}{EF}\text{（上比下 = 上比下）}∴BCAB​=EFDE​（上比下 = 上比下） ∴ABAC=DEDF（上比全 = 上比全）∴\frac{AB}{AC} = \frac{DE}{DF}\text{（上比全 = 上比全）}∴ACAB​=DFDE​（上比全 = 上比全） ∴BCAC=EFDF（下比全 = 下比全）∴\frac{BC}{AC} = \frac{EF}{DF}\text{（下比全 = 下比全）}∴ACBC​=DFEF​（下比全 = 下比全）

相似三角形​

定义​

在 △ABC\triangle ABC△ABC 和 △A′B′C′\triangle A'B'C'△A′B′C′ 中，
如果 ∠A=∠A′∠A = ∠A'∠A=∠A′，∠B=∠B′∠B = ∠B'∠B=∠B′，∠C=∠C′∠C = ∠C'∠C=∠C′，ABA′B′=BCB′C′=ACA′C′=k\frac{AB}{A'B'} = \frac{BC}{B'C'} = \frac{AC}{A'C'} = kA′B′AB​=B′C′BC​=A′C′AC​=k，
即三个角分别相等，三条边成比例，
我们就说 △ABC\triangle ABC△ABC 与 △A′B′C′\triangle A'B'C'△A′B′C′ 相似，相似比为 kkk。
相似用相似符号“∽∽∽”表示，读作“相似于”。
△ABC\triangle ABC△ABC 与 △A′B′C′\triangle A'B'C'△A′B′C′ 相似，记作 △ABC∽△A′B′C′\triangle ABC∽\triangle A'B'C'△ABC∽△A′B′C′。

判定​

预备定理​

平行于三角形一边的直线和其他两边相交，所构成的三角形与原三角形相似。

几何语言​
∵DE//BC∵DE//BC∵DE//BC ∴△ADE∽△ABC∴\triangle ADE∽\triangle ABC∴△ADE∽△ABC

判定定理 AAAAAA​

几何语言​
∵∠A=∠A′,∠B=∠B′∵∠A = ∠A', ∠B = ∠B'∵∠A=∠A′,∠B=∠B′ ∴△ABC∽△A′B′C′∴\triangle ABC ∽ \triangle A'B'C'∴△ABC∽△A′B′C′

判定定理 SASSASSAS​

两边成比例且夹角相等的两个三角形相似。

几何语言​
∵ABA′B′=ACA′C′,∠A=∠A′∵\frac{AB}{A'B'} = \frac{AC}{A'C'}, ∠A = ∠A'∵A′B′AB​=A′C′AC​,∠A=∠A′ ∴△ABC∽△A′B′C′∴\triangle ABC ∽ \triangle A'B'C'∴△ABC∽△A′B′C′

判定定理 SSSSSSSSS​

三边成比例的两个三角形相似。

几何语言​
∵ABA′B′=BCB′C′=ACA′C′∵\frac{AB}{A'B'} = \frac{BC}{B'C'} = \frac{AC}{A'C'}∵A′B′AB​=B′C′BC​=A′C′AC​ ∴△ABC∽△A′B′C′∴\triangle ABC ∽ \triangle A'B'C'∴△ABC∽△A′B′C′
