注意：在 Unicode 中， ASCII 字符块被称作 U+0000..U+007F 基础拉丁（Basic Latin）。

示例​

运行此代码

#include <iostream>
int main()
{
    std::cout << "Printable ASCII:" << std::endl;
    for (char i = 32; i < 127; ++i) {
        std::cout << i << ' ';
        if (i % 16 == 15)
            std::cout << std::endl;
    }
}
